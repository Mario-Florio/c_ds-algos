Compile:
```bash
sh compile.sh
```

Run:
```bash [cmds]
./main.exe
```
cmds:
- test [args]

Running test alone will run all test suites. For specific test suites:
- Linked List Node [llnode]
- Linked List [ll]
- Stack [stack]
- Queue [queue]
- Binary Search Tree Node [bstnode]
- Binary Search Tree [bst]
